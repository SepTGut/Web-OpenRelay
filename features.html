Fitur Unggulan

Semua yang Kamu Butuhkan

OpenRelay hadir dengan fitur lengkap dari firmware hingga dashboard — semua dalam satu paket open source siap pakai.

⚡
128 Relay Channel
Ekspansi tak terbatas menggunakan PCF8574 dan MCP23017 via I2C. Satu ESP32 untuk semua kebutuhan otomasi rumah atau industri.
↳ MAX_RELAYS = 128
📡
MQTT Native
Kontrol relay dari mana saja via MQTT broker publik atau privat. Dukungan WSS untuk dashboard GitHub Pages yang di-host secara HTTPS.
↳ PubSubClient v2.8
🌐
Web Dashboard
Dashboard responsif di GitHub Pages dengan HTTPS + MQTT WSS. Akses dari browser tanpa instalasi apapun — lokal maupun remote.
↳ GitHub Pages + HTTPS
🔧
PCB Custom
Desain PCB menggunakan EasyEDA dengan dukungan I2S audio (INMP441), power management MSMP2307DN + AP2114H-3.3, dan ekspander GPIO.
↳ EasyEDA Design
💾
Konfigurasi Persisten
Semua pengaturan relay, sensor, dan config tersimpan di LittleFS dan bertahan setelah restart atau update firmware via OTA.
↳ LittleFS 8MB
🔄
OTA Update
Update firmware via jaringan WiFi tanpa kabel. Password OTA terkonfigurasi untuk keamanan. Hostname relaycrtl.
↳ Password: relay_ota
🎞️
Pattern Sequencer
Buat urutan nyala/mati relay dengan timing kustom per langkah. Cocok untuk efek lampu, otomasi pabrik, dan demonstrasi kompetisi.
↳ Steps + Repeat + Delay
📊
Sensor Digital
Baca status sensor digital dan publikasikan ke MQTT secara real-time. Integrasi langsung dengan relay untuk otomasi berbasis kondisi.
↳ /sensor/N/v topic
🛡️
Watchdog & Failsafe
Hardware watchdog 10 detik dan allOff() single-batch untuk keamanan. Relay tidak terjebak kondisi ON saat crash atau hang.
↳ esp_task_wdt 10s

Detail Teknis

Struktur packet protokol dan konfigurasi pin PCB yang dikonfirmasi.

relay-arch.cpp C++
// Tiga layer relay control writeRelay(n, on) // HW direct write setRelayState(n, on) // update + dirty flag setRelay(n, on) // state + flush allOff() // single batch safe // Default config activeLow = true MAX_RELAYS = 128 mqttBuf[2560] mqtt.setBufferSize(2048)
pin-config.h C++
// I2C Bus SDA = IO21 | SCL = IO22 // MCP23017 Interrupt INTA = IO32 | INTB = IO33 MIRROR = OFF // separate A/B // I2S Microphone (INMP441) SCK=IO23 WS=IO14 SD=IO13 // Status LEDs IO16, IO17, IO25, IO26
platformio.ini INI
[env:esp32] platform = espressif32 framework = arduino board_build.partitions = default_8MB.csv lib_deps = ESPAsyncWebServer-esphome ^3.1.0 ArduinoJson ^7 PubSubClient ^2.8.0 build_flags = -DCONFIG_ASYNC_TCP_RUNNING_CORE=1

Relay Write Layer — urutan eksekusi

Fungsi Aksi Kapan Digunakan Contoh
writeRelay() Hardware direct write Internal low-level saja GPIO / expander bit
setRelayState() Update state + dirty flag Bulk update tanpa flush allOff() loop
setRelay() State + flush + MQTT pub Single relay dari API/UI POST /relay/on
allOff() Batch off + single pub Failsafe / reset semua Watchdog trigger
