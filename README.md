# OpenRelay

> Open source ESP32-based IoT relay controller platform for FIKSI 2026.

![OpenRelay Banner](https://septgut.github.io/Web-OpenRelay/)

## Overview

OpenRelay is a standalone IoT relay control ecosystem built around ESP32.
The project combines:

- Firmware
- Web dashboard
- MQTT communication
- PCB hardware
- Remote relay control
- WLED ecosystem compatibility

Designed for:

- Smart home automation
- Small industry automation
- Educational IoT projects
- Research & development
- SMK innovation competition projects

---

## Features

- ESP32-based relay controller
- Up to 128 relay channels
- MQTT support
- REST API support
- WebSocket local control
- WLED-compatible ecosystem
- LittleFS configuration storage
- OTA firmware updates
- PCF8574 + MCP23017 expander support
- Responsive web dashboard
- GitHub Pages deployment
- Hardware watchdog failsafe
