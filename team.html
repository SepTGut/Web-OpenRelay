Kompetisi

FIKSI 2026

Festival Inovasi dan Kreativitas Siswa Indonesia — ajang kompetisi nasional untuk pelajar SMK di bidang teknologi dan inovasi.

FIKSI (Festival Inovasi dan Kreativitas Siswa Indonesia) adalah kompetisi nasional bergengsi bagi siswa SMK untuk menampilkan karya terbaik di bidang teknologi, seni, dan kewirausahaan.

OpenRelay diajukan pada bidang Teknologi Informasi dan Komunikasi, menggabungkan tiga komponen utama: firmware ESP32, desain PCB custom, dan web dashboard berbasis GitHub Pages.

Proyek ini dirancang sepenuhnya sebagai karya mandiri dengan pendekatan open source — semua komponen dapat dikembangkan dan direplikasi oleh komunitas luas.

Timeline Proyek
Tahap 1
Riset & Perencanaan
Studi ekosistem WLED, analisis protokol MQTT, dan perancangan arsitektur sistem relay berbasis ESP32.
✓ Selesai
Tahap 2
Pengembangan Firmware
Firmware ~1585 baris, 15 bug diperbaiki. Fitur MQTT, WebSocket, OTA, LittleFS, watchdog, dan pattern sequencer.
✓ Selesai
Tahap 3
Desain PCB
PCB custom EasyEDA dengan ekspander MCP23017, INMP441 I2S, power management. 3 item schematic masih dalam perbaikan.
⟳ Hampir Selesai
Tahap 4
Web Dashboard
Dashboard remote via GitHub Pages dengan HTTPS + MQTT WSS. UI lokal tersimpan di LittleFS ESP32.
✓ Selesai
Tahap 5
Dokumen FIKSI
BMC, TOWS Matrix, dokumen teknis lengkap. Beberapa bagian masih dalam proses finalisasi.
⟳ Dalam Proses
Tahap 6
Pengujian & Presentasi
Pengujian hardware akhir, perakitan PCB, dan persiapan presentasi untuk juri FIKSI 2026.
