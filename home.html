Apa Itu OpenRelay?

Sistem kontrol relay open source berbasis ESP32 untuk otomasi rumah, industri, dan eksperimen IoT modern.

OpenRelay mendukung MQTT, REST API, WebSocket, OTA update, serta ekspansi relay menggunakan PCF8574 dan MCP23017.

Firmware ESP32 custom
Dashboard modern berbasis web
Kompatibel dengan ekosistem WLED
Mendukung hingga 128 relay
