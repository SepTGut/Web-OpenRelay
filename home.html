Apa Itu OpenRelay?

Sistem kontroler relay IoT open source yang dibangun di atas ESP32, dirancang untuk memudahkan otomasi rumah dan industri dengan antarmuka modern.

OpenRelay adalah proyek firmware, PCB, dan web dashboard terintegrasi untuk mengontrol relay secara lokal maupun jarak jauh melalui jaringan internet.

Dibuat sebagai proyek standalone (bukan WLED usermod) untuk kompetisi FIKSI 2026, OpenRelay mengadopsi identitas WLED sehingga kompatibel dengan ekosistem yang sudah ada.

Sistem mendukung ekspansi GPIO menggunakan IC PCF8574 dan MCP23017, memungkinkan hingga 128 relay dari satu unit ESP32 tanpa hardware tambahan yang mahal.

Semua konfigurasi disimpan di LittleFS dan relay dapat dikontrol via MQTT, WebSocket lokal, maupun REST API bawaan.

Semua yang Kamu Butuhkan

OpenRelay hadir dengan fitur lengkap dari firmware hingga dashboard, semua dalam satu paket open source.

⚡
128 Relay Channel
Ekspansi tak terbatas menggunakan PCF8574 dan MCP23017 melalui I2C. Satu ESP32 untuk semua kebutuhan.
📡
MQTT Native
Kontrol relay dari mana saja via MQTT broker publik atau privat. Dukungan WSS untuk dashboard GitHub Pages.
🌐
Web Dashboard
Dashboard responsif berbasis GitHub Pages dengan HTTPS + MQTT WSS. Akses dari browser tanpa instalasi apapun.
🔧
PCB Custom
Desain PCB menggunakan EasyEDA dengan dukungan I2S audio (INMP441), power management, dan ekspander GPIO.
💾
Konfigurasi Persisten
Semua pengaturan (relay, sensor, config) tersimpan di LittleFS dan bertahan setelah restart atau update firmware.
🔄
OTA Update
Update firmware via jaringan WiFi tanpa perlu kabel. Password OTA dapat dikonfigurasi untuk keamanan.
🔌
Pattern Sequencer
Buat urutan nyala/mati relay dengan timing kustom. Cocok untuk efek lampu, otomasi pabrik, dan demonstrasi.
📊
Sensor Digital
Baca status sensor digital dan kirim ke MQTT secara real-time. Integrasi langsung dengan relay untuk otomasi.
🛡️
Watchdog & Failsafe
Hardware watchdog 10 detik dan allOff() single-batch untuk keamanan. Relay tidak terjebak dalam kondisi ON saat crash.

Kontrol via MQTT

OpenRelay menggunakan protokol MQTT standar, kompatibel dengan Home Assistant, Node-RED, dan aplikasi IoT lainnya.

// Publish Topics
home/relay/v STATUS
home/relay/relay/N/v PER RELAY
home/relay/g BRIGHTNESS
home/relay/c COLOR
home/relay/status LWT
// Broker & Koneksi

Mulai Sekarang

Dashboard web bisa langsung diakses dari browser. Firmware dan PCB tersedia untuk dikembangkan lebih lanjut.

Dashboard web siap pakai yang dihosting di GitHub Pages. Akses dan kontrol relay dari browser menggunakan MQTT over WSS. Tidak butuh instalasi.
